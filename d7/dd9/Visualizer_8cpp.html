Detailed Description

Implements visualization modes for the raytracer using SFML.

Author
Noé Carabin
Version
1.0
Date
2025-05

This file provides rendering logic for real-time visualization of raytracing results, using various rendering modes:

  • SCANLINE: renders the image line by line, top to bottom.
  • TILES: renders the image in square tiles (default 64×64).
  • RANDOM: renders the pixels in a shuffled, randomized order.

The rendering process uses OpenMP for parallel computation and optionally displays progress via a colored progress bar. It supports interactive display and updates through SFML, and polls window events to ensure responsiveness.

This module is only compiled if the USE_SFML macro is defined.
