Detailed Description

A node (internal or leaf) of the bounding-volume hierarchy.

An internal node stores two children (left, right), while a leaf stores a small array of primitive pointers.
The hit method first tests the node box; on success it traverses the children (or the primitives) and returns the closest intersection.
