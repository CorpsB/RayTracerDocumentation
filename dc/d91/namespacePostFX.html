Execute the full post-processing pipeline on a frame-buffer.

Parameters
fbFlat vector of RGB pixels, row-major, size = w·h.
w,hResolution in pixels.
contrastContrast factor (see adjustContrast).
saturationSaturation factor (see adjustSaturation).
vignetteStrengthStrength of the vignette (see vignetteFactor).

Each pixel is processed in place: c → contrast → saturation → toneMap → vignette.

Parameters
cLinear-space HDR RGB colour.
Returns
Tone-mapped RGB colour clamped to [0,1].

Current implementation uses the ACES™ approximation c / (c + 1) (can be replaced by a more accurate curve later).
