Detailed Description

Declares functions to load scenes from configuration files (.cfg) into a Scene object.

Author
Cyrian PITTALUGA
Version
1.0
Date
2025-05

These functions parse LibConfig-based configuration files and populate a Scene object with cameras, primitives, lights, and materials. The recursive loader supports include or nested file references, while the base loader parses a single scene file.
