Detailed Description

Declares the axis-aligned bounding box (AABB) and the BVH node used for ray-tracing acceleration.

Author
Cyrian Pittaluga
Version
1.0
Date
2025-05

This header provides:

  • the AABB structure with a fast “slab” intersection routine,
  • the helper function surrounding_box(),
  • the BVHNode class plus the buildBVH() factory that assembles a balanced tree from a list of primitives.
