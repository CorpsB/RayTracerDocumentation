Detailed Description

Implementation of STL (Stereolithography) file loading for 3D mesh import in the raytracer.

Author
Noé Carabin
Version
1.0
Date
2025-05

This file provides the implementation of a loader that reads both binary and ASCII STL files and constructs a Mesh object from the parsed triangles.

Features:

  • Automatic detection of STL format (binary or ASCII)
  • Parses triangles and converts coordinates into the raytracer's coordinate system
  • Centers the mesh geometry around the origin
  • Supports coordinate swapping for Y/Z consistency with internal representation

Key functions:

  • STLLoader::load() – entry point that returns a centered Mesh from an STL file
  • Internal helpers:
    • isBinarySTL() – determines whether the STL is in binary format
    • loadBinary() – reads and parses a binary STL
    • loadAscii() – reads and parses an ASCII STL
    • centerMesh() – adjusts all triangles to center the mesh at (0,0,0)
