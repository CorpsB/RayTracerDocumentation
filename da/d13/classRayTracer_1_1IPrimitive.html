Detailed Description

Pure interface that every geometric primitive must implement.

Provides the essential functions needed by the raytracing engine: ray intersection test and bounding box query.

Returns the axis-aligned bounding box of the primitive.

This is used by the acceleration structures such as BVH to spatially partition the scene.

Ray–primitive intersection test.

Checks whether a given ray intersects the primitive, and if so, fills the intersection record with details (point, normal, etc.).

Parameters
rayThe incoming ray in world space.
t[out] Ray parameter at the intersection point (smallest positive t).
rec[out] Intersection record (position, normal, material, etc.).
Returns
true if the ray intersects the primitive.
