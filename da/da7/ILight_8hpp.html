Detailed Description

Abstract interface for all light sources in the path tracer.

This header defines the pure-virtual base class ILight used to represent different types of light sources. Subclasses provide implementations for ambient, directional, point, and other light types by overriding the methods declared here:

  • type() : returns the lightâ€™s enum type;
  • intensity() : returns the scalar power of the light;
  • direction() : returns the normalized direction vector from a hit point;
  • position() : returns the world-space position of the light (if applicable).
