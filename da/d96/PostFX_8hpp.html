Detailed Description

Declares simple post-processing helpers (contrast, saturation, tone-mapping, vignette) for the ray-tracer frame-buffer.

Author
Cyrian Pittaluga
Version
1.0
Date
2025-05

All functions operate on linear-space RGB values stored in a flat frame-buffer (std::vector<Math::Vector3D>). The default pipeline executed by PostFX::postProcess performs:

  1. Contrast adjustment,
  2. Saturation adjustment,
  3. Filmic tone-mapping,
  4. Radial vignette. Each step can be enabled / disabled via its scaling parameter.
